# Portfolio

Personal portfolio website for Sonu Verma with a static frontend and Express.js backend API served from the same Render web service.

## Run Locally

```bash
npm install
npm start
```

Open:

```text
http://localhost:3002
```

API base:

```text
http://localhost:3002/api
```

## Deploy on Render

Use this repository as a Render Web Service.

- Build Command: `npm install`
- Start Command: `npm start`
- Node version: `>=18`

The Express server serves the frontend files and backend API from the same deployed link.
